## Overview

An interactive **data visualization project** analyzing road accident data across London using **Tableau**. The dashboard reveals patterns in accident frequency, severity, and geographic distribution, providing actionable insights for urban safety planning and traffic management.

---

## Live Dashboard

🔗 **View on Tableau Public:** [London Accident Analysis Dashboard](https://public.tableau.com/app/profile/srivatsa.g1779/viz/LondonAccidentAnalysis/Story1)

---

## Key Features

- **Geographic heat maps** — Visualize accident hotspots across London boroughs
- **Temporal analysis** — Identify trends by time of day, day of week, and season
- **Severity breakdown** — Filter by accident severity (fatal, serious, slight)
- **Interactive filters** — Drill down by date range, weather conditions, and road type
- **Story-driven narrative** — Multi-page Tableau story guiding through key findings
